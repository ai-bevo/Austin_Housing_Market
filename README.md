## Background <a name="background"></a>

The Austin Housing market is one of the hottest markets in 2021, and these listings/collected data show how that market has changed over the past couple of years. For this project we created a visualization dashboard by utilizing a Python Flask-powered API, HTML/CSS, JavaScript, and a SQL database. poltly, leaflet, D3 and charts.js. 


# Final Production Database
